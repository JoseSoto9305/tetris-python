### INSTALLATION

<br> 

The codes have been tested with Python 3.6 and 3.7 on Debian based linux distributions (Mint 19, Ubuntu 16).<br>
You only need pygame installed on your computer:

```
pip install pygame
```
---

### Usage

To run the game just type:
```
python tetris.py
```
